## ğŸŒ Data Source Description
The data used in this project is sourced from a **Weather API**, which provides both current and forecasted weather information. The API returns semi-structured data that includes environmental parameters such as temperature, humidity, wind speed, visibility, air quality index (AQI), pollution metrics (PM10, PM2.5, CO, NO2, SO2), rain probability, and sunrise and sunset timings.

Working with API data required careful handling to ensure the data was properly structured and ready for analysis within Power BI.
